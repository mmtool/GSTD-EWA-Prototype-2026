# Style Review — EWA 3.0 Enterprise Fintech Command Center

## Key Findings
1. Design reads as generic enterprise admin template, not distinctive EWA 3.0 fintech product
2. Typography is serviceable but flat — page titles, numbers, filters all similar generic rhythm
3. Cards use thin outlined defaults — need sharper financial-console aesthetic
4. No signature motif — need a recurring EWA-specific visual idea (ledger/flow/trust)
5. Product voice is functional but generic — needs more authoritative financial tone

## Amendments to Apply
- KPI cards: Remove teal borders, use solid navy top accent strip, heavier number weight (font-bold text-3xl), tabular nums
- Sidebar: Already navy — good. Keep teal active state.
- Page titles: Use uppercase tracking for module titles, lighter weight for subtitles
- Signature motif: Add a subtle horizontal "ledger flow line" divider between sections
- Table headers: Use all-caps tracking-wider, slightly lighter color
- Numbers: font-mono font-bold tabular-nums throughout for financial data
- Module frame: "Command Center" terminology instead of generic labels
- Card surfaces: Remove default border shadows, use clean white with subtle bottom border only
- Date badge: Format as "12 JUL 2026" uppercase
- Status badges: Use compact dot+text instead of full outline badges where possible
